(квадрат из n строк, в каждой строке n звездочек)
n - гарантируется, что целое число

Вывести на экран фигуру n*n (только для нечетного n) из звездочек:
*   .   *   .   *
.   *   *   *   .
*   *   *   *   *
.   *   *   *   .
*   .   *   .   *


 n = 5

if n % 2 == 0:
    print("Ошибка! Число n должно быть нечетным.")
else:
    for i in range(n):
        for j in range(n):
            if i % 2 == 0:
                if j % 2 == 0:
                    print("*", end=" ")
                else:
                    print(".", end=" ")
            else:
                if j % 2 == 0:
                    print(".", end=" ")
                else:
                    print("*", end=" ")
        print()
